# Logistic Regression Case Study

## Overview

This project uses logistic regression to a classification problem using a structured machine learning workflow. The goal is to build, tune, and evaluate a model that can predict outcomes based on input features.

---

## Steps Completed

### 1. Data Preparation

* Defined feature matrix `X` and target variable `y`
* Split data into training and test sets

### 2. Model Training

* Build baseline Logistic Regression model
* Trained the model using the training data

### 3. Model Evaluation

* Evaluated performance using accuracy
* Analyzed with confusion matrix and classification report

### 4. Cross-Validation

* Used cross-validation to get a more reliable performance estimate
* Compared results on multiple data splits

### 5. Hyperparameter Tuning

* Tuned the regularization parameter `C`
* Performed manual search and GridSearchCV
* Selected the best model based on CV score

### 6. Final Model

* Trained final model using best parameters
* Evaluated performance on unseen test data

---

## Key Takeaways

* Cross-validation improves model reliability
* Accuracy alone is not always sufficient for evaluation
* Hyperparameter tuning helps balance model complexity and performance
